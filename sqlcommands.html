📘 SQL Commands Introduction

1. SQL Command Types

  • DDL (Data Definition Language) → CREATE, ALTER, DROP
  • DML (Data Manipulation Language) → INSERT, UPDATE, DELETE
  • DQL (Data Query Language) → SELECT

2. Create & Drop Database

CREATE DATABASE company; DROP DATABASE company;

3. Data Types

  • INT → Integer
  • VARCHAR → Text
  • DATE → Date values
  • FLOAT → Decimal values

4. Key Attributes

  • Primary Key: Unique + Not NULL
  • Foreign Key: References another table
  • Parent Table: Referenced table
  • Child Table: Referencing table
FOREIGN KEY (dept_id) REFERENCES department(id);

5. DML Commands

INSERT

INSERT INTO emp (id, name) VALUES (1, 'A');

UPDATE

UPDATE emp SET salary = 5000 WHERE id = 1;

DELETE

DELETE FROM emp WHERE id = 1;

6. DQL - SELECT with WHERE

SELECT * FROM emp WHERE salary > 3000;

7. Operators with WHERE

  • =, >, <, >=, <=, !=
  • AND, OR, NOT
SELECT * FROM emp WHERE salary > 3000 AND dept_id = 1;

8. LIMIT

SELECT * FROM emp LIMIT 5;

9. Compound Search Condition

SELECT * FROM emp WHERE salary > 3000 AND dept_id = 1 OR name LIKE 'A%';

👉 Uses multiple conditions together

10. Missing Data (NULL)

SELECT * FROM emp WHERE salary IS NULL; SELECT * FROM emp WHERE salary IS NOT NULL;

11. ALTER Table

ALTER TABLE emp ADD age INT; ALTER TABLE emp DROP COLUMN age;

12. Simple Date Functions

SELECT CURRENT_DATE; SELECT YEAR(date), MONTH(date) FROM orders;

13. Summary

  • DDL → Structure (CREATE, ALTER, DROP)
  • DML → Data changes (INSERT, UPDATE, DELETE)
  • DQL → Data retrieval (SELECT)
  • WHERE filters data
  • NULL represents missing values
