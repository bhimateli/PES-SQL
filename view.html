📘 SQL Views - Quick Notes

1. What is a View?

A virtual table based on a SQL query.

CREATE VIEW v_emp AS SELECT * FROM employees;

2. Types of Views

  • Simple View
  • Complex View
  • Joined View
  • Group View

3. Joined Views

View created using multiple tables.

CREATE VIEW emp_dept AS
SELECT e.name, d.dept_name
FROM emp e JOIN dept d ON e.dept_id = d.id;

4. Horizontal & Vertical Views

  • Horizontal: Filters rows (WHERE)
  • Vertical: Selects specific columns

5. Row & Column Subset View

Combination of row + column filtering.

CREATE VIEW v_subset AS
SELECT name, salary FROM emp WHERE salary > 5000;

6. Group View

Uses aggregate functions.

CREATE VIEW dept_salary AS
SELECT dept_id, SUM(salary) FROM emp GROUP BY dept_id;

7. CHECK OPTION

Ensures inserted/updated data satisfies view condition.

CREATE VIEW v_high_salary AS
SELECT * FROM emp WHERE salary > 5000
WITH CHECK OPTION;

8. DROP VIEW

Deletes a view.

DROP VIEW v_emp;
