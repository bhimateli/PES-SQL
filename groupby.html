📘 GROUP Functions & GROUP BY

1. GROUP Functions (Aggregate Functions)

These functions perform calculations on multiple rows and return a single value.

  • COUNT() → Counts number of rows
  • AVG() → Calculates average value
  • MAX() → Returns highest value
  • MIN() → Returns lowest value
  • SUM() → Returns total sum
SELECT COUNT(*) FROM employees;

2. GROUP BY with Aggregate Function

GROUP BY is used to divide data into groups and apply aggregate functions on each group.

SELECT dept_id, AVG(salary) FROM employees GROUP BY dept_id;

👉 Calculates average salary for each department

3. Grouped Queries

Queries that use GROUP BY to summarize data.

SELECT dept_id, COUNT(*) AS total_employees FROM employees GROUP BY dept_id;

👉 Shows number of employees per department

4. Aggregation with GROUP BY

Multiple aggregate functions can be used together.

SELECT dept_id, SUM(salary), MAX(salary) FROM employees GROUP BY dept_id;

👉 Total and maximum salary per department

5. Multiple Grouping Columns

You can group data using more than one column.

SELECT dept_id, job_role, AVG(salary) FROM employees GROUP BY dept_id, job_role;

👉 Groups data by department and job role

6. NULL Values in GROUP BY

  • NULL values are treated as a separate group
  • All NULL values are grouped together
SELECT dept_id, COUNT(*) FROM employees GROUP BY dept_id;

👉 Employees with NULL dept_id will form one group

7. Important Rules

  • All non-aggregated columns must be included in GROUP BY
  • GROUP BY is executed after WHERE
  • HAVING is used to filter grouped results

8. Summary

  • GROUP functions calculate values over multiple rows
  • GROUP BY divides data into groups
  • Supports multiple columns for grouping
  • NULL values form their own group
